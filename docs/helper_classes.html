class helper_classes.slurm_status(value)

Slurm job status enumerator.

created = 1

job files have been created and stored locally

submitted = 2

jobs has been submitted to slurm on remote host

done = 3

job finished successfully (all gaussian steps finished) and have been retrieved from host

failed = 4

job failed

incomplete = 5

job is incomplete, it should be resubmitted

uploaded = 6

job has been uploaded to the DB succesfully
