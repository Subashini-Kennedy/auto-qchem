descriptor_functions module

descriptor_functions.occupied_volume(geometry_df, atom_idx, r, mesh_density=30) → float

Compute occupied volume fraction within a sphere of radius ‘r’ for an atom at position ‘atom_idx’. Each atom radius is taken to be its Van der Waals radius.

Parameters

  • geometry_df (pd.DataFrame) – geometry dataframe, must contain ‘X’, ‘Y’, ‘Z’ and ‘AN’ (atomic number) columns

  • atom_idx (int) – index of the atom to use as ‘central’ atom

  • r (float) – occupied volume radius in Angstroms

  • mesh_density (int) – density of the mesh for numerical integration (MAX=100)

Returns

float, occupied volume fraction
