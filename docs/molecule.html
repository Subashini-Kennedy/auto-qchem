class molecule.molecule(smiles, num_conf, engine='rdkit', rdkit_ff='MMFF94', ob_gen3D_option='best')

Wrapper class for molecule

__init__(smiles, num_conf, engine='rdkit', rdkit_ff='MMFF94', ob_gen3D_option='best') → None

Initialize the molecule with a conformational ensemble

Parameters

  • smiles (str) – SMILES string

  • num_conf (int) – maximum number of conformations to generate

  • engine (str) – conformation search engine (rdkit or openbabel)

  • rdkit_ff (str) – rdkit supported force-field to use when engine=’rdkit’

  • ob_gen3D_option (str) – option to use with openbabel gen3D for search of initial geometry
