gaussian_input_generator class

class gaussian_input_generator.gaussian_input_generator(molecule, workflow_type, directory, theory, solvent, light_basis_set, heavy_basis_set, generic_basis_set, max_light_atomic_number)

Generator of Gaussian input files

__init__(molecule, workflow_type, directory, theory, solvent, light_basis_set, heavy_basis_set, generic_basis_set, max_light_atomic_number)

Initializes Gaussian input generator for a given molecule.

Parameters

  • molecule (molecule) – molecule object

  • workflow_type (str) – Gaussian workflow type, allowed types are: ‘equilibrium’ or ‘transition_state’

  • directory (str) – local directory to store input files

  • theory (str) – Gaussian supported Functional (e.g., B3LYP)

  • solvent (str) – Gaussian supported Solvent (e.g., TETRAHYDROFURAN)

  • light_basis_set (str) – Gaussian supported basis set for elements up to max_light_atomic_number (e.g., 6-31G*)

  • heavy_basis_set (str) – Gaussin supported basis set for elements heavier than max_light_atomic_number (e.g., LANL2DZ)

  • generic_basis_set (str) – Gaussian supported basis set for generic elements (e.g., gencep)

  • max_light_atomic_number (int) – maximum atomic number for light elements
